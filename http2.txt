1. Простий GET-запит
Створи функцію fetchUserData(), яка виконує GET-запит до API для отримання даних користувача за 
URL: https://jsonplaceholder.typicode.com/users/1. Після отримання відповіді, 
виведи ім'я користувача у консоль.

2. POST-запит для додавання нового користувача
Напиши функцію createUser(), яка відправляє POST-запит для створення нового 
користувача за допомогою URL: https://jsonplaceholder.typicode.com/users. 
Дані нового користувача мають виглядати так:

{
  name: "John Doe",
  email: "johndoe@example.com"
}
Після успішного запиту, виведи ID створеного користувача у консоль.

3. Асинхронне очікування декількох запитів
Створи функцію fetchMultipleUsers(), яка паралельно виконує GET-запити для 
отримання даних про трьох користувачів (ID 1, 2, 3) з https://jsonplaceholder.typicode.com/users. 
Використовуй Promise.all(), щоб обробити всі відповіді одночасно. Виведи імена всіх користувачів
у консоль після отримання відповідей.

4. Обробка помилок у запитах
Напиши функцію fetchPost(), яка виконує GET-запит до API за допомогою 
URL: https://jsonplaceholder.typicode.com/posts/101. Якщо такого посту 
не існує, виведи у консоль повідомлення "Пост не знайдено". Використовуй 
метод catch для обробки можливих помилок.

5. Запит з обробкою статус-кодів
Напиши функцію deleteUser(), яка видаляє користувача з ID 5 за допомогою DELETE-запиту
до URL: https://jsonplaceholder.typicode.com/users/5. Після успішного видалення, перевір
статус-код відповіді. Якщо код 200, виведи повідомлення "Користувача видалено". 
Якщо інший код, виведи "Помилка видалення".
