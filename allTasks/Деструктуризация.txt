Задача 1: Основы деструктуризации
Дан объект:

javascript
Копировать код
const person = {
  name: "Alice",
  age: 25,
  city: "New York"
};
Задание: Используя деструктуризацию, создайте переменные name, age и city, чтобы получить значения этих свойств из объекта.

Задача 10: Деструктуризация и вычисляемые имена свойств
Дан объект:

javascript
Копировать код
const dimensions = {
  length: 20,
  width: 10,
  height: 5
};
Задание: Используя деструктуризацию, создайте переменные volume и surfaceArea, вычислив объем и площадь поверхности объекта (предполагается, что это прямоугольный параллелепипед). Формулы для расчета:

Объем: length * width * height
Площадь поверхности: 2 * (length * width + length * height + width * height)
