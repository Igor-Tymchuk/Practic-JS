1. Заполнение формы по умолчанию из localStorage
Задача: При загрузке страницы заполните поля формы значениями, сохраненными в localStorage. Если данных в localStorage нет, оставьте поля пустыми.
Совет: Используйте метод localStorage.getItem().

2. Сохранение данных формы в localStorage при отправке
Задача: После отправки формы сохраняйте введенные данные в localStorage, чтобы при следующем открытии страницы поля были заполнены этими значениями.
Совет: Используйте событие submit и метод localStorage.setItem().

3. Сброс данных формы и очистка localStorage
Задача: Добавьте кнопку "Reset", которая очищает все поля формы и удаляет соответствующие записи из localStorage.
Совет: Используйте методы localStorage.removeItem() и localStorage.clear().

4. Проверка сохраненных данных в sessionStorage
Задача: При заполнении формы сохраните введенные данные в sessionStorage в реальном времени (при каждом изменении поля). После перезагрузки страницы форма должна восстанавливать значения из sessionStorage.
Совет: Используйте событие input и метод sessionStorage.setItem().

5. Автозаполнение формы на основе предыдущего ввода
Задача: Автоматически заполните поле "First Name" на основе предыдущего ввода пользователя, сохраненного в localStorage, когда он начнет вводить свои данные.
Совет: Используйте события focus или input.

6. Подсчет и отображение оставшихся символов в поле
Задача: Добавьте ограничение на количество символов в поле "Last Name" и отобразите пользователю количество оставшихся символов в реальном времени.
Совет: Используйте атрибут maxlength и событие input.

7. Запрет отправки формы без подтверждения пароля
Задача: Проверьте, совпадают ли значения полей "Password" и "Confirm Password" при отправке формы. Если они не совпадают, предотвратите отправку формы и отобразите сообщение об ошибке.
Совет: Используйте событие submit и метод preventDefault().

8. Отправка данных формы с подтверждением через alert
Задача: Перед отправкой формы покажите пользователю alert с подтверждением данных (например, "Are you sure?"). Если пользователь соглашается, форма отправляется, иначе отправка отменяется.
Совет: Используйте метод confirm().

9. Сохранение выбранного пола в localStorage
Задача: Сохраните выбранный пользователем пол (radio button) в localStorage и при следующем посещении страницы автоматически выберите соответствующую опцию.
Совет: Используйте localStorage.setItem() и localStorage.getItem().

10. Создание и использование счетчика посещений страницы
Задача: Создайте счетчик, который сохраняется в localStorage и увеличивается при каждом посещении страницы. Отображайте количество посещений пользователю.
Совет: Используйте localStorage.getItem() и localStorage.setItem() для отслеживания и обновления счетчика.
